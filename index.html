Simplifique UIs JavaScript dinâmicas com o padrão Model-View-View Model (MVVM)

{% include download-button.html %}

Conceitos chave

  • Bindings declarativos

    Associar facilmente elementos DOM com dados do model usando uma sintaxe legível e concisa

  • Atualização automática da UI

    Quando o estado dos dados do seu model são alterados, sua UI atualiza automaticamente

  • Dependency Tracking

    Configura implicitamente correntes de relações entre model data, para transformar e combinar

  • Templating

    Gera rapidamente UIs sofisticadas e aninhadas como uma função para o seu model data
